## 🚀 Quick Actions

### Just Starting?
```
1. npm run dev                    # Start server
2. http://localhost:3000         # Open browser
3. Upload a resume + job desc    # Test it
4. Review results                # See scoring
```

### Want to Test Thoroughly?
```
1. Open QUICKSTART.md or TESTING.md
2. Download/create sample PDFs from test-data/
3. Run through testing scenarios
4. Check scoring accuracy
```

### Want to Deploy?
```
1. Read DEPLOYMENT.md
2. Choose Vercel or another platform
3. Connect GitHub repo
4. Deploy in 1 click!
```

### Want to Customize?
```
1. Open src/lib/scoringEngine.ts
2. Adjust weights (line ~165)
3. Run: npm run build
4. Test: npm run dev
5. Deploy when ready
```
